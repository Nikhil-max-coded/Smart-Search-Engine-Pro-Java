# Smart Search Engine Pro Using Java

## Overview

Smart Search Engine Pro is an advanced Java-based search engine that efficiently searches text documents using an Inverted Index.

## Features

- Single Keyword Search
- Multi Keyword Search (AND/OR)
- Inverted Index
- Ranking Engine
- Suggestion Engine
- Search History
- Word Frequency Analysis
- Statistics
- Console Interface

## Technologies

- Java
- Object-Oriented Programming
- Java Collections Framework
- HashMap
- ArrayList
- File Handling

## How to Run

Compile

```
javac *.java
```

Run

```
java Main
```
